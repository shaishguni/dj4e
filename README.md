# Local Setup

```bash
git clone https://github.com/shaishguni/dj4e.git
```
```bash
virtualenv venv && source venv/bin/activate
```
```bash
pip install -r requirements.txt
```
```bash
cd dj4e
```
```bash

python manage.py migrate
```
```bash
python manage.py runserver
```


After that you good to go.
