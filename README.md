# Local Setup

```json
git clone https://github.com/shaishguni/dj4e.git
```
```json
virtualenv venv && source venv/bin/activate
```
```json
pip install -r requirements.txt
```
```json
cd dj4e
```
```json

python manage.py migrate
```
```json
python manage.py runserver
```


After that your are good to go.
